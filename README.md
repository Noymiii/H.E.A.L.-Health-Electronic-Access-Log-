# H.E.A.L.-Health-Electronic-Access-Log-
A web-based Electronic Health Record (EHR) system designed to streamline hospital workflows through digital patient management.
H.E.A.L. solves the problem of fragmented medical records by centralizing patient data. It features a **serverless architecture** (using Google Sheets/Apps Script as a backend), making it lightweight and easy to deploy without dedicated server costs.
